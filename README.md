#  MyHeroDex

MyHeroDex es una aplicación móvil Android desarrollada en **Java** con **Android Studio**, que consume la **API oficial de Marvel** para mostrar información detallada de héroes y cómics.

Este proyecto fue desarrollado **de forma individual** como **parcial final** de la asignatura **Aplicaciones Móviles**.

---

## Funcionalidades

###  Dashboard (Home)
- Estadísticas generales obtenidas desde la API de Marvel
- Información dinámica y actualizada

###  Héroes
- Búsqueda de héroes
- Imagen del personaje
- Descripción y origen
- Información detallada

###  Cómics
- Búsqueda de cómics
- Imagen y detalles
- Precio del cómic

###  Autenticación
- Inicio de sesión
- Cierre de sesión

---

##  Tecnologías usadas

- **Lenguaje:** Java  
- **IDE:** Android Studio  
- **Arquitectura:** Activities / Fragments  
- **Consumo de API:** Marvel API  
- **Diseño:** XML  
- **Control de versiones:** Git & GitHub
